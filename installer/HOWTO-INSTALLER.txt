ZubCut — Windows installer
==========================

This installer build:
  - Installs to:  Program Files\ZubCut
  - Settings in:  %APPDATA%\ZubCut\
  - Executable:   ZubCut.exe
  - Autostart:    registry value "ZubCut"

One-time setup
--------------
1. Python 3.8+ and project deps:
     pip install -r requirements.txt
     pip install pyinstaller pillow

2. Inno Setup 6 (free):
     https://jrsoftware.org/isdl.php

3. Optional but recommended for first-time installs:
     Place Npcap installer at:
       installer\npcap-1.87.exe
     The setup will run it silently only when Npcap is missing.

Build the installer
-------------------
From the project root, double-click:

     installer\Build-Installer.bat

Or manually:
     python build.py
     "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer\ZubCut.iss

Output:  output\ZubCut-Setup-1.29.exe   (version matches installer script)

Run the .exe installer as Administrator when prompted.

Branding (optional)
-------------------
The same script builds the installer whether users download it from GitHub or the app runs an update.

  - Already set: dark wizard (WizardStyle=modern dark) and small logo (exe\zubcut_icon.png).
  - Light: add installer\branding\setup.ico and set SetupIconFile in ZubCut.iss (file icon in Explorer / taskbar while setup runs).
  - Medium: add WizardImageFile = left strip ~164×314 PNG for the welcome/finished pages (match dark theme).
  - Heavier: LicenseFile / InfoBeforeFile text, custom WizardStyleFile (VCL styles), or per-page Pascal UI — more time, rarely needed.
