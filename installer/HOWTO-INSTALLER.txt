ArpCut Updated — Windows installer
==================================

This build is separate from the original "ArpCut" release:
  - Installs to:  Program Files\ArpCut Updated
  - Settings in:  %APPDATA%\ArpCutUpdated\
  - Executable:   ArpCutUpdated.exe
  - Autostart:    registry value "ArpCutUpdated" (not "arpcut")

One-time setup
--------------
1. Python 3.8+ and project deps:
     pip install -r requirements.txt
     pip install pyinstaller pillow

2. Inno Setup 6 (free):
     https://jrsoftware.org/isdl.php

Build the installer
-------------------
From the project root, double-click:

     installer\Build-Installer.bat

Or manually:
     python build.py
     "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer\ArpCutUpdated.iss

Output:  output\ArpCutUpdated-Setup-1.1.0.exe   (version matches installer script)

Run the .exe installer as Administrator when prompted. Npcap must still be installed
for capture (same as the original app).
